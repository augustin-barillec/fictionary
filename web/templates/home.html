Fictionary

A slack app to play fictionary.

Set up a game

The organizer types one of the following command:

  • /fictionary freestyle
  • /fictionary english
  • /fictionary french

The first command lets the organizer write a question and an answer.
The other commands let the organizer pick a question from one of the
question banks below.

Question Banks

  • English
  • French

Gameplay

  1. Players see only the question and write a credible answer.
  2. Players vote for the answer they believe is correct.
  3. Players win one point for voting for the game's answer
    and two points for each vote cast for the answer they wrote.
