Privacy Policy

Fictionary is an app for fun. It collects only the data necessary to its functioning.

All data is stored in a private Google Cloud Platform project.

During the installation, the following data is collected:

  • Team ID, e.g. TXXXXXXX. This allows the bot to select the correct authorization token in order to interact with your workspace.
  • Authorization Token, the token granted by Slack to the bot in order to securely communicate with your workspace.

This token is given two permissions scopes:

  • chat:write. It enables the app to post messages in the conversations where it has been invited.
  • commands. It enables the app to provide a slash command which lets a user launch a game.

For each game, the following data is collected:

  • User IDs involved with the game, e.g. UXXXXXX.
  • Channel ID where the game takes place, e.g. CXXXXXX.
  • The question and the answer written by the creator of the game if it is a freestyle game.
  • The answers written by the players.

If a game ends successfully, the data written by the users is deleted in about a day. If a game crashes, it is kept about 3 more days to help for debugging. The data not written by the users is stored for logging and monitoring purposes. It is deleted 30 days after its creation.

If you want the data associated with your workspace to be deleted, you can send an email at augustin.barillec@yokyok.ninja with your team ID.
