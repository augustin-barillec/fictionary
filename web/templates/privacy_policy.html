Privacy Policy

Fictionary is a Slack app for fun. It collects only the data necessary to
its functioning.

All data is stored in a Firestore instance in Europe. It is a database
provided by Google Cloud Platform and it is not public to internet.

During the installation, we collect the following informations:
  • Team ID, ex: TXXXXXXX. This allows the bot to select the
    correct authorization token in order to interact with your workspace.
  • Authorization Token, the token granted by Slack to our bot in
    order to securely communicate with your workspace.
For each game, we collect the following informations:
  • User IDs involved with the game, ex: UXXXXXX.
  • Channel ID where the game takes place, ex: CXXXXXX.
  • Questions and answers written by the users.
If a game ends successfully, its data is deleted automatically within
the hour. If a game crashes, its data is kept to help for debugging.
Once the code is fixed, the game's data is deleted manually.

If you want the data associated with your workspace to be removed, you
can send me a email at augustin.barillec@yokyok.ninja with your team ID. {% endblock %}
