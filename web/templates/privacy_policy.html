Privacy Policy

Fictionary is a Slack app for fun. It collects only the data necessary to its functioning.

All data is stored in a Firestore instance in Europe. It is a private database provided by Google Cloud Platform.

During the installation, we collect the following informations:

  • Team ID, ex: TXXXXXXX. This allows the bot to select the correct authorization token in order to interact with your workspace.
  • Authorization Token, the token granted by Slack to our bot in order to securely communicate with your workspace.

For each game, we collect the following informations:

  • User IDs involved with the game, ex: UXXXXXX.
  • Channel ID where the game takes place, ex: CXXXXXX.
  • The question written by the organizer if it is a freestyle game.
  • The answers written by the players.

If a game ends successfully, the data written by the users is automatically deleted within a day. If a game crashes, it is kept to help for debugging. Once the code is fixed, it is deleted manually. The data not written by the users is stored for monitoring purposes.

If you want the data associated with your workspace to be removed, you can send a email at augustin.barillec@yokyok.ninja with your team ID.
